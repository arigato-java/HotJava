Unable to Launch Viewer
&browser.name.prefix;TM &browser.name.suffix; is unable to launch the application "&appname;" to view the file &url;.

The information for launching applications, (or external content viewers), from &browser.name; is probably not set correctly.

Please check the User's Guide for information on how to set up &browser.name; to launch external content viewers.

To modify your external viewer settings, go to Content Viewer Preferences in the Edit menu.

An external content viewer is an application external to &browser.name; used to view a file.
