Unable to Launch Viewer
&browser.name; is unable to launch the application "&appname;" to view the file &url;.

The information for launching applications, (or external content viewers), from &browser.name; is probably not set correctly.

An external content viewer is an application external to &browser.name; used to view a file.

To do something with this file right now, choose one of the following:

  • Click on Save to save the file to your local file system.
  • Click on View in &browser.name; to attempt to view the file in &browser.name;.
  • Click on View in Application to choose an application to launch in order to view the file you clicked on.

Note: Clicking on any of these buttons is a one-time operation and does not result in any permanent configuration of &browser.name;.
