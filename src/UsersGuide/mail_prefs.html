Mail Preferences

Use the Mail Preferences to specify the return email address you want &browser.name.prefix;TM &browser.name.suffix; to use when sending email. Access these preferences by choosing Edit->General Preferences. Then select Mail in the options on the left side of the General Preferences dialog box.

The preferences on this page are for outgoing mail only. &browser.name; does not provide a mail reader.

Type your Email Address and the name or IP address of your Mail (SMTP) Server.

SMTP stands for Simple Mail Transport Protocol. It is the standard protocol for transferring email between systems on the Internet.

In most cases, either the system you're running &browser.name; on is your mail server, or the server to which your machine is connected is your mail server.

If you don't specify a mail server, &browser.name; looks for a server named "localhost." If that name is not found, it looks for a server named "mailhost." If one of these servers is found, &browser.name; will use that as the mail server.

Check with your system administrator if you're not sure what to specify for your mail server.
