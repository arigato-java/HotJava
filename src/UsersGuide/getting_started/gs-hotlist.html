Creating a Bookmark List

When you discover a web page that you'd like to revisit or think you might visit frequently, you can save the address of the page in a Bookmark List. This enables you to load the page again without having to retype its address. You can have more than one Bookmark List.

To Save a Web Site Address

  • Choose Add Bookmark from the Bookmarks menu.
    &browser.name.prefix;TM &browser.name.suffix; adds the address of the current page to the Personal Bookmarks list.

    Or:

  • Choose File Bookmark from the Bookmarks menu, and choose the list where you'd like the address to be saved from the submenu.

To Revisit the Site

  • If the bookmark (the page title) appears at the bottom of your Bookmarks menu, simply choose the bookmark item to revisit the site. By default, the bookmarks in your Personal Bookmarks list appear at the bottom of the Bookmarks menu.

    Or:

  • Choose Go to Bookmark from the Bookmarks menu, choose the list where the bookmark was saved from the submenu, and then choose the bookmark you want to revisit.

For more information about bookmarks, including information about how to use the Edit Bookmarks window, see &browser.name; Menus.
